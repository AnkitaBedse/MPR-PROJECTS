Quadratic Programming Solver (Interactive GUI Tool)

This application is an interactive tool designed to help users formulate and solve Quadratic Programming (QP) problems through a simple graphical interface. It utilizes the Karush–Kuhn–Tucker (KKT) conditions to compute optimal solutions and provides visual representation for better understanding.
The tool is especially useful for students and learners who want to explore optimization concepts in a practical and intuitive way.

Key Features

• Supports both minimization and maximization quadratic optimization problems
• Easy graphical interface for entering matrices and vectors
• Capable of solving problems with up to four variables and four constraints
• Provides 2D visualization of constraints and optimal solutions for two-variable cases
• Includes a help section that explains how to use the application
• Built using Python, Tkinter, and Matplotlib for a clean and interactive experience

Steps to Use the Application

Specify the number of variables (n) and number of constraints (m).

Click “Generate Matrices” to create the required input fields.

Enter the necessary values for:

Matrix Q (quadratic coefficient matrix)

Vector c (linear coefficients)

Matrix A and Vector b representing constraints in the form Ax ≤ b

Choose the desired optimization type (Minimize or Maximize).

Press “Solve & Plot” to compute the optimal solution and display the graph (for 2-variable problems).

Additional Notes

• Graphical plotting is available only for problems with two variables.
• The solution is obtained using the Karush–Kuhn–Tucker (KKT) optimality conditions.
• This tool is intended for educational purposes, demonstrations, and learning the basics of quadratic programming.
• Developed using Python with Tkinter for the interface and Matplotlib for visualization.
